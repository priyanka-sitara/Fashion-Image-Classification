# Fashion-Image-Classification
# Fashion Image Classification using CNN

This project builds a deep learning model to classify fashion products using the Fashion-MNIST dataset.

Technologies Used:
Python
TensorFlow
Google Colab
NumPy
Matplotlib

Dataset:
Fashion-MNIST (70,000 images of fashion items)

Categories:
T-shirt
Trouser
Pullover
Dress
Coat
Sandal
Shirt
Sneaker
Bag
Ankle boot

Model:
Convolutional Neural Network (CNN)

The model learns patterns from images and predicts the category of clothing items.
